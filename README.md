# PVRPOSE AI Landing Page

AI automation consulting services for service businesses.

## Deployment

This site is deployed on Vercel.

## Technology Stack

- HTML5
- Tailwind CSS (via CDN)
- Vanilla JavaScript
- Google Fonts (Inter)
- Modern UX/UI Design (2026 trends)

## Design Features

### Visual Design
- **Glassmorphism effects** - Frosted glass aesthetic with blur and transparency
- **Gradient accents** - Pink-to-purple gradients matching brand identity
- **Micro-interactions** - Hover effects, smooth transitions, subtle animations
- **Enhanced typography** - Optimized hierarchy with hero headlines
- **White space strategy** - Generous spacing for premium feel
- **Logo integration** - Brand logo with subtle glow animation

### Performance Optimizations
- Optimized meta tags for SEO
- Open Graph and Twitter Card support
- Preloaded critical assets
- Lazy loading for images
- Efficient CSS with Tailwind utilities
- Mobile-first responsive design

### Accessibility
- WCAG 2.1 AA compliant
- Semantic HTML5 structure
- ARIA labels on interactive elements
- Proper heading hierarchy
- High contrast ratios
- Keyboard navigation support

Simply open `index.html` in your browser to view locally.
