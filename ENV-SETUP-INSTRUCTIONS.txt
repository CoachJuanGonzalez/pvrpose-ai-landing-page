================================
VERCEL ENVIRONMENT VARIABLES SETUP
================================

URGENT: Set these environment variables in Vercel Dashboard NOW!

1. Go to: https://vercel.com/dashboard
2. Select project: pvrpose-ai-landing-page
3. Go to: Settings > Environment Variables
4. Add these 3 REQUIRED variables:

-----------------------------------
VARIABLE 1: AIRTABLE_API_KEY (REQUIRED)
-----------------------------------
Name: AIRTABLE_API_KEY
Value: [Your Airtable Personal Access Token]
Environments: ✓ Production ✓ Preview ✓ Development

How to get:
1. Go to https://airtable.com/create/tokens
2. Click "Create new token"
3. Name it: "PVRPOSE AI Landing Page"
4. Add scopes:
   - data.records:read
   - data.records:write
5. Click "Add a base" and select your AI Assessments base
6. Click "Create token"
7. Copy the token (starts with "pat...")

-----------------------------------
VARIABLE 2: AIRTABLE_BASE_ID (REQUIRED)
-----------------------------------
Name: AIRTABLE_BASE_ID
Value: [Your Airtable Base ID]
Environments: ✓ Production ✓ Preview ✓ Development

How to get:
1. Open your Airtable base in browser
2. Look at the URL: https://airtable.com/appXXXXXXXXXXXXXX/...
3. Copy the part starting with "app" (e.g., app1234567890abcd)

-----------------------------------
VARIABLE 3: AIRTABLE_TABLE_NAME (REQUIRED)
-----------------------------------
Name: AIRTABLE_TABLE_NAME
Value: tblAssessmentSubmissions
Environments: ✓ Production ✓ Preview ✓ Development

Important: Use the TABLE ID (starts with "tbl"), NOT the table name!

How to get:
1. Open your Airtable base
2. Click on the table
3. Look at URL: https://airtable.com/appXXX/tblYYYYYYYYYYYYYY/...
4. Copy the part starting with "tbl" (e.g., tblAssessmentSubmissions)

-----------------------------------
AIRTABLE TABLE STRUCTURE REQUIRED:
-----------------------------------
Your Airtable table must have these fields (exact names):

- first_name (Single line text)
- email (Email)
- score (Number)
- assessment_answers (Long text)
- timestamp (Date with time)
- source (Single line text)
- consent_marketing (Checkbox)
- consent_timestamp (Date with time)
- jurisdiction (Single line text)
- fingerprint (Single line text)
- status (Single select: New, Contacted, Converted)
- top_priority (Long text)

-----------------------------------
AFTER SETTING VARIABLES:
-----------------------------------
1. Go to Vercel Dashboard > Deployments
2. Click "..." on latest deployment > Redeploy
3. Wait for deployment to complete
4. Test the assessment form at: https://pvrpose.ai/assessment.html
5. Check Airtable for new record
6. Check Vercel Functions logs if it fails:
   Vercel Dashboard > Deployment > Functions tab

-----------------------------------
TROUBLESHOOTING:
-----------------------------------
If submission fails:
1. Check Vercel function logs for errors
2. Verify all 3 environment variables are set
3. Verify Airtable API token has correct permissions
4. Verify table ID (tblXXX) not table name
5. Verify all required fields exist in Airtable table
